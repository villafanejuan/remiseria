# Remisería - Sistema de Gestión

Sistema de gestión para remisería con panel de admin y app de remiseros.

## Características

- **Admin**: Gestión de remiseros, pasajeros, viajes y reportes
- **Remisero**: Registro de viajes, búsqueda de pasajeros, control de estado y pagos
- **Viajes**: Local y Larga Distancia
- **Pagos**: Efectivo y Transferencia

## Requisitos

- PHP 8+
- MySQL / MariaDB
- XAMPP o similar

## Instalación

1. Importar `database.sql` en MySQL
2. Configurar `config/database.php` con credenciales
3. Acceso: `http://localhost/remiseria/`
